# Object Oriented Programming — C++

A collection of C++ programs written as university assignments,
covering the core principles of Object Oriented Programming.

---

## Concepts Covered

- Classes and Objects
- Constructors and Destructors
- Encapsulation (private / public)
- Inheritance
- Polymorphism
- Operator Overloading
- Templates
- Design Patterns
